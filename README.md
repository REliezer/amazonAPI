#### üìù Setup de Base de Datos

   ```sql
       @firstName VARCHAR(255),

       @admin BIT = 0
   AS
   BEGIN
       INSERT INTO amazon.users (email, firstName, lastName, active, admin)
       VALUES (@email, @firstName, @lastName, @active, @admin);
       
       SELECT SCOPE_IDENTITY() as user_id;
   END;
   ```
